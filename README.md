# Age Calculator

A simple Python program that calculates a person's age in different time units.

## Features

- Calculates age in months
- Calculates age in days
- Calculates age in hours
- Calculates age in seconds
- Handles leap years

## Example

**Input**

```text
16
```

**Output**

```text
16 years
192 months
5844 days
140256 hours
504921600 seconds
```

## Technologies

- Python 3
- time
- calendar
